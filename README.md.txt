# ❌⭕ Tic-Tac-Toe (Terminal Game)

A terminal-based Tic-Tac-Toe game written in C.

## 🎮 Features
- Supports 3×3, 4×4, and 5×5 board sizes
- Two players take turns (x and o)
- Win detection for rows, columns, and diagonals
- Tie detection
- Move history saved to `moves.txt` with timestamp
- Special rule on 3×3: oldest move removed after 3rd move per player

## ▶️ How to Run
```bash
gcc tictactoe.c -o tictactoe
./tictactoe
```
